# Sanduíches da Janice - Backend

## Diagrama de Entidade e Relacionamento (DER)

Use a ferramenta [drawdb](https://github.com/drawdb-io/drawdb) para visualizar ou editar o DER.

> Você pode clonar o repositório e rodar a ferramenta localmente.

`Crie um arquivo em branco no drawdb` e então `importe o arquivo` [docs/DER.json](docs/DER.json).

> Se lembre de `exportar como JSON` para este repositório e commitar, caso você altere a modelagem.
